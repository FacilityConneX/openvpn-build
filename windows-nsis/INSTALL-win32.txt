GENERAL QUICKSTART FOR WINDOWS

FCXTunnel requires the a client configuration file to be placed in C:\Program Files\FCXTunnel\config\ directory. You should 
obtain this file from the QuickConneX UI or FacilityConneX Support.

The FCXTunnel runs in the backgrond as a Windows service called FCXTunnelService. It is recomended that this service is set to "Automatic Start" with Recovery options set as "Restart the Service" for all failure attempts.

For any more questions, please contact support@facilityconnex.com
